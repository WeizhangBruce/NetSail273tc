網路航海時代・讀我

由於「網路航海時代」的作者仍不成熟，安裝設定可能會比較困難。

請在理解這一點之後，再嘗試安裝。

  • 什麼是網路航海時代

    這是一款以大航海時代為背景的 CGI 網路模擬遊戲。
    你可以以成為海賊王、冒險王或交易王為目標，也可以建設城鎮等來讓港口更加繁榮，是一款能自由遊玩的模擬遊戲。


  • 使用規定

    請務必閱讀「Net Castle 腳本使用規定」。
    下載本程式者，視為已同意使用規定。

  • 腳本設定

    sea.cgi、new.cgi、list.cgi、setting.cgi 需要修改。
    腳本內有說明，基本上請查看內容後進行設定。
    這裡特別提醒幾個重要部分。

    首先在 sea.cgi、new.cgi、list.cgi 中，
    第一行的
    #!/usr/bin/perl
    請依照伺服器環境修改。
    通常是　#!/usr/bin/perl　或　#!/usr/local/bin/perl　其中之一。

    在 setting.ini 中，
    $adps = '0123';
    請變更「0123」。這是管理員密碼。

    $usrdir = 'usrdir';
    $datadir = 'data';
    $citydir = 'city';
    請依照各自的目錄設定分別修改。

    $lockkey = 2;
    這是設定是否進行檔案鎖定。如果設為 2 無法正常運作，請嘗試設為 1。


    若有設置公會制度，且想在個人資料中顯示所屬公會，
    請替換為「for guild.LZH」中的 intro.cgi 與 list.cgi。

  • 應用

    冒險情報、船隻、交易品等，可透過修改 data 資料夾中的資料
    自由新增或刪除。
    首先說明冒險情報。

    例如查看 cow.dat 的內容，會看到
    1009,115,4,「水牛？再往北一點吧」
    1009,116,1,發現了水牛,水牛的角
    如下所示。
    各行的意義為：
    區域編號,港口編號,地點編號,訊息(,寶物)
    港口編號等請參照區域資料（1001～1009.dat）。
    區域資料各行開頭即為港口編號。
    地點編號為 1=港口、2=交易品店、3=造船所、4=酒館、5=城鎮、6=襲擊。
    當位於該港口編號港口中的指定地點編號位置時，會顯示訊息。
    只有最後一行可以再加入一個要素，
    若有此要素，便會作為寶物取得。
    也就是說，在這個例子中，抵達「哈瓦那」的「港口」瞬間，便會取得「水牛的角」。
    建議在訊息中寫下前往下一個訊息顯示地點的提示。
    請參考各種資料後再進行調整。
    若要建立新的冒險情報資料，檔名可任意命名（但不可使用數字），副檔名請設為 .dat。
    接著，需要在「a_list.dat」中新增資料。
    1009<><>1400<>cow.dat<>「這片大陸的北方，有一種強壯的牛喔。」<>
    應該會有類似這樣的一行。其意義為：
    情報販售區域<>情報販售港口<>價格<>檔名<>購買時顯示的訊息<>
    （最後一定要加入 <>！）
    請以同樣方式，配合新建立的冒險情報加以新增。

    接著是船隻，這裡僅說明資料格式。
    船隻資料存放於 item.dat。
    バルシャ<>1<>balsha.gif<>1001<><>50<>70<>8<>1200<>
    第一行如上所示。其意義如下：
    名稱<>（船隻時為 1）<>圖片檔案<>販售區域<>販售港口<>載貨量<>HP<>速度<>價格<>

    關於交易品，區域資料中的各行分別對應各港口。
    這裡也僅說明資料格式。
    001<>里斯本<>38,-9<>岩鹽,30△橄欖油,100△火槍,200△大砲,300△銀,280△金,550△古董品,300<>1
    其意義為：
    港口編號<>名稱<>北緯（南緯時為 -）,東經（西經時為 -）<>販售貨物名稱1,價格1△販售貨物名稱2,價格2（以 △ 分隔輸入）<>物價指數<>
    如下所示。
