ネット航海時代は作者が未熟であるため、設置は難しいかもしれません。

そのことをご了承の上、設置を試みてください。

  • ネット航海時代とは

    大航海時代を背景としたCGIのネット・シミュレーションゲームです。
    海賊王・冒険王・交易王を目指すもよし、町などを作って港を盛り上げるも良し、自由にプレイできるシミュレーションゲームです。

    必ず「Net Castle スクリプト利用規定」をお読み下さい。

  • スクリプトの設定

    sea.cgi、new.cgi、list.cgi、setting.cgiは変更する必要があります。
    スクリプト内に説明がありますので基本的には中を見て設定してください。
    ここでは特に重要な部分を注意しておきます。

    まずsea.cgi、new.cgi、list.cgiにおいて
    一行目の
    #!/usr/bin/perl
    をサーバに合わせて変更してください。
    通常は　#!/usr/bin/perl　か　#!/usr/local/bin/perl　のどちらかです。

    setting.iniにおいて
    $adps = '0123';
    「0123」を変更してください。管理人パスワードです。

    $usrdir = 'usrdir';
    $datadir = 'data';
    $citydir = 'city';
    それぞれ各自のディレクトリ設定に従って変更してください。

    $lockkey = 2;
    ファイルのロックをするかどうかです。2で上手く作動しない場合1を試してください。


    ギルド制度を設けていて、プロフィールに所属ギルドを表示させたい場合は、
    「for guild.LZH」に入っているintro.cgiとlist.cgiに差し替えてください。

  • 応用

    冒険情報、船、交易品等はデータフォルダのデータを変更することで
    自由に追加・削除できます。
    まず、冒険情報について説明します。

    例えばcow.datの中を覗くと
    1009,115,4,「バッファロー？もうちょっと北だね」
    1009,116,1,バッファローを発見しました,バッファローの角
    となっています。
    各行の意味は、
    エリアナンバー,港ナンバー,ポイントナンバー,メッセージ(,財宝)
    となっています。
    港ナンバー等はエリアデータ（1001～1009.dat）を参照してください。
    エリアデータの各行の先頭が港ナンバーです。
    ポイントナンバーは1=港、2=交易品屋、3=造船所、4=酒場、5=町、6=襲撃、です。
    港ナンバーの港の、ポイントナンバーの場所にいると、メッセージが表示されます。
    最後の行にのみ、もう一つ要素を付け加えることが出来て、
    これがある場合、財宝として入手することになります。
    つまり、この例の場合、「ハバナ」の「港」に着いた瞬間、「バッファローの角」が入手されます。
    メッセージには次のメッセージ表示ポイントへのヒントを書くと良いでしょう。
    いろいろなデータを見て、検討してみてください。
    新しく冒険情報のデータを作成する場合、ファイル名は何でも良いので（ただし数字は×）拡張子を.datにして下さい。
    そして、「a_list.dat」にデータを追加する必要があります。
    1009<><>1400<>cow.dat<>「この大陸の北の方には、力強い牛がいるよ。」<>
    という行があると思います。これは
    情報販売エリア<>情報販売港<>価格<>ファイル名<>買ったときに表示されるメッセージ<>
    となっています。（最後に<>を必ず入れてください！）
    同じようにして新しく作った冒険情報に合わせて追加してください。

    次に船ですが、データの説明だけしておきます。
    船のデータが入っているのはitem.datです。
    バルシャ<>1<>balsha.gif<>1001<><>50<>70<>8<>1200<>
    一行目は以上のようになっています。意味は以下の通りです。
    名前<>（船の場合は1）<>画像ファイル<>販売エリア<>販売港<>積載量<>HP<>速度<>価格<>

    交易品に関してはエリアデータの各行が各港に対応しています。
    これもデータの説明だけしておきます。
    001<>リスボン<>38,-9<>岩塩,30△オリーブ油,100△鉄砲,200△大砲,300△銀,280△金,550△骨董品,300<>1
    これは
    港ナンバー<>名前<>北緯（南緯の場合は-）,東経（西経の場合は-）<>取り扱い積荷名1,価格1△取り扱い積荷名2,価格2（△で区切って入れる）<>物価指数<>
    となっています。
