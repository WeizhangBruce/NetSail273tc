===================== setting.cgi的內部設定解說 =====================
#!/usr/bin/perl
視伺服器的設定而改變
通常都是　#!/usr/bin/perl　或　#!/usr/local/bin/perl

$adps = '0123';
「0123」一定要更改！那是管理員的密碼

$usrdir = 'usrdir';
$datadir = 'data';
$citydir = 'city';
當資料夾更名之後，這裡也要更改成你更名之後的資料夾名稱

==========================自由追加==================================
遊戲中的「交易品」、「船艦」、「冒險情報」均可自行增加、刪除

=======================追加新冒險情報===============================
我們先做增加冒險任務的教學
舉例：ostrich.dat
1004,052,4,「關於那個腳很長大鳥的事情嗎？<br>看到後也許會讓你嚇一跳的。」
1004,052,6,發現了駝鳥,駝鳥的蛋

各行的意思說明：
1004,052,		代表著城市的編號，請參閱(1001～1009.dat）
1004,052,"4",		帶表城鎮內的設施
『海上＝0、港口＝1、交易所＝2、造船場＝3、酒場＝4、城鎮＝5、襲擊＝6』
另外..最後一行「1004,052,6,發現了駝鳥,駝鳥的蛋」是任務最終的結尾
句子最後的「鴕鳥的蛋」為寶物

整體來說，任務的架構就是
『海域代碼,城市代碼,設施代碼,情報訊息,寶物』
寶物只有寫在冒險情報最後一段才有作用(可加可不加)

當編輯完之後，請存檔成*.dat
請自行摸索~
======================追加新冒險情報酒館列表========================
上面只是冒險情報的寫法，還不會出現在酒館的冒險情報內
這時你必須編輯：a_list.dat

舉例：
<>001<>400<>fsta.dat<>「交易品店的老闆總覺得知道某些事情。」<>
1001<><>250<>pisa.dat<>「去地中海可以看見比薩斜塔。」<>

解說：
<>001<>400<>fsta.dat<>「交易品店的老闆總覺得知道某些事情。」<>
<>城市代碼<>價格<>冒險情報.dat<>初始情報訊息<>

1001<><>250<>pisa.dat<>「去地中海可以看見比薩斜塔。」<>
海域代碼<><>價格<>冒險情報.dat<>初始情報訊息<>

Q:上面這兩段有什麼不同
A:第一段式獨立任務，只有指定的城市才購買得到
第二段則是共同任務，同一海域均可購買得到

請自行摸索~

=============================追加新船艦=============================
編輯：item.dat
輕木帆船<>1<>balsha.gif<>1001<><>50<>80<>7.5<>1200<>1<>2<>
名字<>船艦均是1<>圖像<>販賣的海域(註1)<>販賣的城市(註2)<>積載量<>耐久<>速度<>價格<>
(註1、2，請對照上面「追加新冒險情報酒館列表」的解說)
